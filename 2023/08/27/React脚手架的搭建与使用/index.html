React脚手架是开发现代Web应用的必备，其充分利用Webpack、Babel、ESlint等工具辅助项目的开发，当然这些工具也无需手动配置即可使用，脚手架的意义更多的是关注的是业务而不是工具的配置；项目的整体技术架构为：react + webpack + es6 + eslint。使用脚手架开发项目的特点：模块化、组件化、工程化。
