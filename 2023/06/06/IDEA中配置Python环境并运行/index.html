1 前言

mac 一般自带 python环境 ，在终端中 输入python就会有相关环境信息

Window 平台上安装 Python 的简单步骤如下

  • 打开 WEB 浏览器访问https://www.python.org/downloads/windows/

  • 选择其中的一个版本下载，下载后，双击下载包，进入 Python 安装向导，安装非常简单，你只需要使用默认的设置一直点击”下一步”直到安装完成即可

2 IDEA 中安 Python 插件

3 创建python工程

在IDEA里，选择file -> new project

对于 windos 环境，若已经下载好python.exe,idea会自动识别，然后一直点击next ，最后设置项目名，finish
