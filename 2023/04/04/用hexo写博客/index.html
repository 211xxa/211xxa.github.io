用hexo写博客

Created2023-04-04|Updated2023-04-29
|Word count:523|Reading time:1min|Post View:

使用 Hexo 写博客可以让你更加专注于内容创作，而不必过多关注博客网站的技术细节和设计问题。以下是使用 Hexo 写博客的基本步骤：

1、安装 Hexo：访问 Hexo 官方网站，根据官方文档的指引，安装 Node.js 以及 Hexo。

2、初始化 Hexo：使用命令行工具进入你想要存储 Hexo 博客的目录，运行 hexo init 和 npm init 命令，这将会初始化一个 Hexo 博客，并创建必要的文件和文件夹。

3、配置 Hexo：在项目根目录下的 _config.yml 文件中配置博客的基本信息，例如博客标题、描述、作者、主题等等。

4、创建新文章：运行 hexo new <文章标题> 命令来创建一篇新的博客文章。创建完成后，会在 source/_posts/ 文件夹中生成一个以文章标题命名的 Markdown 文档。

5、编辑文章内容：使用 Markdown 语法，将你的文章内容填充到所创建的 Markdown 文档中。

6、发布文章：使用 hexo clean 和 hexo generate 命令来生成静态网站，并使用 hexo deploy 命令将静态网站上传到云服务器或者部署到 GitHub Pages 上。

关于具体格式，配置和编写 Markdown 格式的 Hexo 文章，包括文章标题、发布日期、标签、分类等信息，可以在你的 Markdown 文件中添加 front matter 部分。一个典型的 front matter 如下：
